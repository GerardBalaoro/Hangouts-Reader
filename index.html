Upload Hangouts.json

Visit Google Takeout then click "Select none" and only check the box next to Hangouts. Within a few minutes or so you will recieve a zip file with Hangouts.json inside it. Extract the file and choose it above. If you have a very large chat history it can take a few minutes to load the file. Don't worry, everything here is done clientside so there's no risk of your chat history being sent anywhere.
